Interactions

DRAG VERTEX
Amber circles are draggable handles. While dragging, if within 30m of a road/building edge, vertex magnetizes to nearest point. Blue highlight shows snap target.
CLICK MIDPOINT
Gray circles on edge midpoints. Click to insert a new vertex at that position. Becomes a draggable amber handle.
RIGHT-CLICK VERTEX
Context menu with "Delete vertex". Disabled if only 3 vertices remain (minimum for polygon).
SNAP TOGGLE
Blue indicator in toolbar. Toggle on/off. When on, dragged vertices magnetize to nearby OSM features (roads, buildings, water).

Implementation: New Component PolygonEditor.tsx

Replaces the current broken editMode state with a proper map-integrated editor. Receives map ref from TerritoryDetail. Manages its own GeoJSON source + layers for vertex handles, midpoints, and polygon preview. On save, calls updateTerritoryBoundaries() which runs the auto-fix pipeline server-side.

Neighbor overlay: When edit mode activates, fetches neighboring territories (already loaded in TerritoryMap via listTerritories()) and renders them as translucent polygons so the user can see adjacent boundaries.
