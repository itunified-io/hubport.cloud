Key Design Decisions

Quick Actions
  • One-tap outcomes on map popup + address list
  • Building colors reflect status (green=visited, red=DNC, amber=not home)
  • Address list auto-sorts by GPS proximity
  • Already-visited + DNC addresses dimmed/struck-through
GPS + Privacy
  • Blue dot + heading cone + accuracy circle
  • Per-publisher "Share" toggle (independent)
  • Service work group: leader creates, publishers join, leader ends
  • Overseer dashboard: sees all sharing publishers
