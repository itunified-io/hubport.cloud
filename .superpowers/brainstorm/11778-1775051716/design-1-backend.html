1. Prisma Schema Addition

schema.prisma — new model
model TerritoryBoundaryVersion {
id String @id @default(uuid())
territoryId String
version Int // 1, 2, 3... per territory
boundaries Json // GeoJSON snapshot of boundary at this version

}

2. postgis-helpers.ts — Spatial Functions

hub-api/src/lib/postgis-helpers.ts — 4 core functions
// 1. Validate + auto-heal geometry
validateGeometry(geojson) → valid GeoJSON
SQL: ST_AsGeoJSON(ST_MakeValid(ST_GeomFromGeoJSON($1)))
// 2. Clip to congregation boundary
clipToCongregation(geojson) → clipped GeoJSON | null
SQL: ST_AsGeoJSON(ST_Intersection(
ST_GeomFromGeoJSON($1),
ST_GeomFromGeoJSON(congregation.boundaries)
))
Returns null if no congregation boundary exists (skip step)
// 3. Clip away neighbor overlaps
clipToNeighbors(geojson, excludeTerritoryId) → { clipped, removedFrom[] }
For each neighbor territory WHERE id != excludeTerritoryId:
geojson = ST_Difference(geojson, neighbor.boundaries)
Returns list of territories that were clipped from
// 4. Detect remaining overlaps (informational)
detectOverlaps(geojson, excludeTerritoryId) → OverlapInfo[]
SQL: SELECT t.id, t.number, t.name,
ST_Area(ST_Intersection(...)::geography) as overlapAreaM2
FROM "Territory" t WHERE ST_Intersects(...) AND t.id != $excludeId

4. API Endpoints

New + modified endpoints
// NEW — Preview auto-fix without saving
POST /territories/:id/preview-fix
Body: { boundaries: GeoJSON }
Returns: { original, clipped, applied[], overlaps[], geometryModified }
No DB write — dry run only
// NEW — Preview for new territory (no ID yet)
POST /territories/preview-fix
Body: { boundaries: GeoJSON }
Same pipeline but excludeTerritoryId = null
// MODIFIED — Now runs auto-fix pipeline + creates version
PUT /territories/:id
If boundaries changed: run pipeline → save clipped → create version
Response adds: autoFix { applied[], overlaps[], geometryModified }
// MODIFIED — Now runs auto-fix pipeline + creates version
POST /territories
If boundaries present: run pipeline → save clipped → create v1
// NEW — List boundary versions
GET /territories/:id/versions
Returns: version history (id, version, changeType, changeSummary, createdAt)
Does NOT include boundaries JSON (too large) — fetch via restore preview
// NEW — Preview + restore a version
POST /territories/:id/restore
Body: { versionId: uuid }
Runs auto-fix on the old geometry → preview → if accepted, saves + new version
// NEW — Detect violations across all territories
GET /territories/violations
Returns: [{ territoryId, number, name, violations: ["exceeds_boundary", "overlaps_302"] }]
Used by map view to show warning badges
