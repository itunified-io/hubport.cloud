🚀 Feature-Phasen
  • 📍
    Treffpunkt-CRUD Phase 1
    Erstellen, bearbeiten, löschen. Karte + Adresse. Wochentag/Uhrzeit.
  • 📅
    Predigtdienstgruppen-Planung Phase 1
    Wochenansicht, Termine erstellen, Publisher-Anmeldung, Gebietszuordnung.
  • 📢
    Aktionen umziehen Phase 1
    Campaigns von Gebiete → Predigtdienst verschieben. Route + Sidebar.
  • 🔐
    RBAC für neue Entitäten Phase 1
    Neue Permissions + Rollenzuweisung (siehe Tabelle links).
  • 🔔
    Push Notifications Phase 2
    Service Worker + Web Push API. Erinnerung vor Diensttermin, Kampagnen-Updates.
  • ✨
    General Polish Phase 2
    UI-Verbesserungen, Animationen, Loading States, Empty States.
  • 📊
    Dienstberichte Phase 3
    Teilnahme-Statistiken, Häufigkeitsauswertung, CSV Export.
